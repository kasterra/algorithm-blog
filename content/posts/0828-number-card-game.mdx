꾸준히 글을 적어야 블로그의 의미가 있을 것 입니다. 본관에도 글을 올리고, 분관인 이곳에도 꾸준히 글을 올려야겠지요.

오늘의 문제는 어제의 그것보다 오히려 더 간단한 문제일 것 입니다. `min`, `max` 파이썬 함수의 사용법을 안다면 간단히 풀 수 있는 문제입니다.

# 오늘 푼 문제 : 숫자 카드 게임

숫자 카드 게임은 여러 개의 숫자 카드 중에서 가장 높은 숫자가 쓰인 카드 한 장을 뽑는 게임이다. 단, 게임의 룰을 지키며 카드를 뽑아야 하고 룰은 다음과 같다.

1. 숫자가 쓰인 카드들이 N * M 형태로 놓여있다. 이때 N은 행의 개수를 의미하며, M은 열의 개수를 의미한다.
2. 먼저 뽑고자 하는 카드가 포함되어 있는 행을 선택한다.
3. 그다음 선택된 행에 포함된 카드들 중 가장 숫자가 낮은 카드를 뽑아야 한다.
4. 따라서 처음에 카드를 골라낼 행을 선택할 때, 이후에 해당 행에서 가장 숫자가 낮은 카드를 뽑을 것을 고려하여 가장 높은 숫자의 카드를 뽑을 수 있도록 전략을 세워야 한다.

카드들이 N * M 형태로 놓여있을 떄, 게임의 룰에 맞게 카드를 뽑는 프로그램을 만드시오.

## 입력조건
- 첫째 줄에 숫자 카드들이 놓인 행의 개수 N과 열의 개수 M이 공백을 기준으로 하여 각각 자연수로 주어진다 (1 ≤ N,M ≤ 100)
- 둘째 줄 부터 N개의 줄에 걸쳐 각 카드에 적힌 숫자가 주어진다. 각 숫자는 1 이상 10,000 이하의 자연수이다.
## 출력조건
- 첫째 줄에 게임의 룰에 맞게 적힌 숫자를 출력한다.

## 예시

### 입력 예시 1
```text
3 3
3 1 2
4 1 4
2 2 2
```

### 출력 예시 1
```text
2
```

### 임력 예시 2
```text
2 4
7 3 1 8
3 3 3 4
```

### 출력 예시 2
```text
3
```

# 나의 풀이

```python
N, M = map(int, input().split())

board = []

for i in range(N):
  board.append(list(map(int, input().split())))

min_list = []

for i in range(N):
  min_num = 987654321
  for number in board[i]:
    min_num = number if number < min_num else min_num
  min_list.append(min_num)

print(max(min_list))
```

이 문제도 어렵지 않습니다. 입력을 받고, 각 줄마다 최소가 무엇인지 알아낸 다음, 그 최소 중에서 가장 큰 것을 고르는 것 입니다. `? :` 문법과 유사한 inline if 문법을 떠올릴 수 있었고, iterate 할 수 있는 요소들에 대해서 `min`, `max`를 사용할 수 있다는 것 또한 다시 떠올릴 수 있었습니다.

여기에서는 사실 게임 보드 전체를 저장할 필요는 없지만, 향후 2차원 배열을 사용할 문제들이 계속 있을 것 이기에, 이차원 배열을 초기화 하는 코드 스니펫을 간단히 수록해 보고자 합니다.

```python
N, M = 3, 4  # 3행 4열
array = [[0] * M for _ in range(N)]
```

파이썬 원라이너는 참으로 대단하지만, 역시 외우고는 있어야 한 것 같습니다. 리스트 컴프리헨션은 오랜만에 보니 참 새롭군요. 조건 제시법이라고 생각하면 다시 익숙해지기 좋은 것 같습니다.
